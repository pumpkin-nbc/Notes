# Docker数据卷

> 卷技术

1. 将容器内的目录，挂载到Linux上
2. 使容器持久化以及同步操作
3. 容器间数据共享

## 使用数据卷

> 方式一：使用命令来挂在

1. `docker run -v`

   ```dockerfile
   # 启动容器
   docker run -itd -v 主机目录:容器内目录 -p 主机端口:容器端口
   ```

2. `docker volume`

   ```dockerfile
   # 查看所有volume的情况
   docker volume ls 
   
   #查看挂载的具体地址
   docker volume inspect 卷名
   
   ```

   

> 匿名挂载

```dockerfile
docker run -itd -v 容器内目录
```

> 具名挂载

```dockerfile
docker run -itd -v 卷名:容器内目录
```
