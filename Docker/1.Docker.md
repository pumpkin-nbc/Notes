> 容器化技术

1. 镜像(image)
   1. 类似模板
   2. 一个镜像可以创建多个容器
2. 容器(container)
   1. Docker独立运行一个或一组应用，通过镜像创建
   2. 支持启动，停止删除等命令
   3. 类似简易的Linux系统
3. 仓库(repository)
   1. 存放镜像的地方
   2. 分为公有仓库，私有仓库
   3. Docker Hub（默认国外）
   4. 国内有容器仓库，或配置镜像加速
