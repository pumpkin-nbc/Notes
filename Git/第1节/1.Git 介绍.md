# 版本控制

---

集中式控制工具

全局变量

`git config --global user.name`

`git config --global user.email  `

1. 初始化本地库

   1. 基本语法

      `git init `

2. 查看本地库状态

   1. `git status`

3. 添加至暂存区

   1. `git add file.name`
   2. 删除暂存区的文件
      1. `git rm --cached file.name`

4. 提交至本地库

   1. `git commit -m '相关信息' file.nam`

5. 查看版本信息

   1. `git reflog` 查看普通日志
   2. `git log` 查看详细日志

6. 恢复历史版本

   1. `git reset --hard 版本号`
