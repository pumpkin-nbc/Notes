# 安装JupyterHub

```shell
# 下载
sudo docker pull jupyter/all-spark-notebook

# 镜像选择
# jupyter/base-notebook
# jupyter/minimal-notebook
# jupyter/r-notebook
# jupyter/scipy-notebook
# jupyter/tensorflow-notebook
# jupyter/datascience-notebook
# jupyter/pyspark-notebook
# jupyter/all-spark-notebook

# 安装
sudo docker run -d --name jupyter -p 8888:8888 --restart=always -e JUPYTER_ENABLE_LAB=yes -v /Jupyter/home/work:/home/jovyan/work -v /etc/localtime:/etc/localtime jupyter/all-spark-notebook
```
